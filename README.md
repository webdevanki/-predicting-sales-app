# Predicting Sales App - ML Forecasting Tool

## Opis
Interaktywna aplikacja do prognozowania sprzedaży oparta na uczeniu maszynowym. Pozwala na analizę trendów historycznych i przewidywanie przyszłych wyników biznesowych na podstawie wgranych plików CSV.

## Funkcje
- **Data Exploration**: Automatyczna wizualizacja trendów i sezonowości.
- **ML Predictions**: Wykorzystanie modeli regresyjnych do estymacji przyszłych przychodów.
- **User-Friendly Dashboard**: Intuicyjny interfejs w Streamlit pozwalający na symulację różnych scenariuszy biznesowych.

## Tech Stack
- Python (Pandas, Scikit-learn, PyCaret)
- Streamlit (Frontend)
- Matplotlib / Seaborn (Wizualizacja)
